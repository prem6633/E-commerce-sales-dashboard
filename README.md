# E-commerce Sales Dashboard (Power BI)

This project presents an interactive Power BI dashboard designed to analyze sales and profit data from an e-commerce business. It focuses on visualizing key metrics such as profit trends by sub-category and performance over time.

## File Included
- `eccomerce sales.pbix`: The main Power BI Desktop file containing the entire dashboard.

## Dashboard Overview

The dashboard provides:
- Profit by Sub-Category: A bar chart showing which product sub-categories are most and least profitable.
- Monthly Profit Trends: A time series breakdown to track changes in profit over months.
- Interactive Filters: Allow users to explore specific sub-categories or time periods.

## Key Features
- Clean and modern layout
- Color-customized visuals for clarity
- Easy-to-read charts and labels
- Designed for business reporting and performance tracking

## Tools & Technologies
- Power BI Desktop (.pbix)
- Data visualization and basic modeling

## Use Cases
- Identify which product lines contribute most to overall profit
- Detect seasonal trends in sales
- Provide actionable insights for business strategy

## How to Use
1. Download the `.pbix` file from this repository.
2. Open it in Power BI Desktop.
3. Use the filters and visuals to explore the data.

## Sample Insights
- Sub-categories like "Printers" and "Bookcases" may show contrasting profitability.
- Certain months show consistent dips or peaks in profit.
